    Instance Method Details

    - (void) initialize() (private)

    Note: Make sure to call TILEDMap.load(path) before instantiating any Entities.

    Initializes the new Entity. Will try to get the map from the mediator.

    - (void) updateViewAndMove(vd) (bound) (private)

    Updates the animationStep counter and viewDirection. If the entity goes into the same direction, only update animationStep If not it will reset the animationStep counter and update the viewDirection

    Parameters:

    • vd (Integer) — the view direction. Can be one of 0, 1, 2 or 3.

    See also:

    • [Entity#viewDirection]

    - (void) onAction(player) (bound)

    Is called if the Player stands in front of this Entity and want's to interact with it.

    Parameters:

    • player (Object) — The Player.

    - (void) update() (bound)

    Is called each tick/frame.

    - (void) moveUp() (bound)

    Note: It won't let @position.y be negative.

    Move the Entity one tile upwards.

    - (void) moveDown() (bound)

    Note: This method relies on mediator.map being set

    Move the Entity one tile downwards. It won't let @position.y be larger than map.numYTiles.

    - (void) moveRight() (bound)

    Note: This method relies on mediator.map being set

    Move the Entity one tile to the right. It won't let @position.y be larger than map.numXTiles.

    - (void) moveLeft() (bound)

    Note: It won't let @position.x be negative.

    Move the Entity one tile upwards.
